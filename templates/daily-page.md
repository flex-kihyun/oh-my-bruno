# Daily Page Template

Notion Daily 페이지의 콘텐츠 구조. morning-briefing 프롬프트가 이 템플릿을 참조하여 페이지를 생성한다.

## 페이지 제목 형식

```
Daily Check-in
```

아이콘: ☀️

## 콘텐츠 구조

```markdown

---
```
